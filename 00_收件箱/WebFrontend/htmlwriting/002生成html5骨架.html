生成 HTML5 骨架

这是一个简单清晰的 HTML 起步指南，教你快速生成页面骨架并预览效果。

一、生成 HTML 骨架的方法

HTML 骨架是网页的基础结构，包含 <!DOCTYPE html>、<html>、<head> 和 <body> 等核心标签。它是构建任何网页的起点，所有内容都基于这个框架展开。

  1. 在英文输入状态下输入 !，然后按 Tab 键即可生成 HTML5 骨架。
  2. 如果未生成，请确认文件已保存并且当前文件类型是 HTML。
  3. 也可以在 VS Code 中使用"生成 HTML5 骨架"命令快速创建结构。

二、预览网页的方式

方法一：直接打开

  1. 直接在文件夹中双击 HTML 文件图标，即可在默认浏览器中打开预览。
  2. 推荐使用 Chrome 浏览器，更适合开发调试和查看效果。
  3. 每次修改后，切换浏览器并手动刷新页面即可查看最新内容。

方法二：Live Server

  1. 在 VS Code 中安装 Live Server 插件，保存文件后自动刷新页面。
  2. 安装完成后，使用快捷键 Ctrl+Shift+P，选择"Open with Live Server"打开页面。
  3. 开发时建议使用分屏模式，一边编辑一边实时预览，效率更高。
