# WebFrontend 文件夹说明

## 📁 文件夹结构

```
WebFrontend/
├── Markdown/          # Markdown 相关资源
│   └── Markdown语法.md  # Markdown 语法指南
└── README.md          # 本说明文档
```

## 📚 文件夹用途

**WebFrontend** 文件夹用于存放与前端开发相关的资源和文档，主要包含以下内容：

- **Markdown 语法指南**：提供 Markdown 基本语法的详细说明，包括外链接的创建方法
- **前端开发参考资料**：后续可添加 HTML、CSS、JavaScript 等前端技术的学习资料
- **项目文档**：可用于存储前端项目的设计文档、需求分析等

## 🎯 核心内容

### Markdown 语法指南
- **位置**：`Markdown/Markdown语法.md`
- **内容**：详细介绍 Markdown 中外链接的创建方法，包括：
  - 行内式链接
  - 带标题的行内式链接
  - 参考式链接
  - 自动链接
  - 核心规则与注意事项

## 🔍 使用建议

1. **查阅语法**：需要使用 Markdown 语法时，可参考 `Markdown/Markdown语法.md` 文件
2. **扩展内容**：可根据需要在本文件夹中添加更多前端相关的资料和文档
3. **保持结构**：建议按照功能模块组织文件，保持文件夹结构清晰
