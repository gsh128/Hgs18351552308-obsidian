# 中长期投资策略分析

## 一、宏观定调（中长期视角）

### 1. 政策面
- **政治局会议（2026.04.28）**：时隔9个月再提"稳定和增强资本市场信心"，政策进入执行兑现期
- **财政/货币政策**："更加积极的财政政策" + "适度宽松的货币政策"，流动性充裕基调不变
- **产业重点**：全面实施"人工智能+"行动、六网新基建、能源资源安全
- **中长期定调**：政策重心从"活跃交易"切换到"信心建设"，着眼长期制度建设

### 2. 经济基本面
- 一季度 GDP +5.0%，达到年度目标上沿
- 一季报全A（剔除金融两油）归母净利润 **+11.5%**，2021 年来首次两位数增长
- PMI 连续处于扩张区间（4月 50.3%）
- 经济复苏基础渐稳，但内需/地产仍在筑底

### 3. 流动性环境
- 国内：央行保持宽松，宏观流动性充裕
- 海外：美联储降息窗口基本关闭，高利率持续
- 北向资金：2026年4-5月持续净流入，中国资产吸引力提升

| 层级 | 细分方向 | 确定性 | 关键催化 |
|:----:|---------|:------:|---------|

- 2026年进入业绩兑现期，从"讲故事"到"看报表"

## 三、中长期持仓框架建议

### 核心-卫星策略

```
┌──────────────────────────────────────────┐
│          核心仓位（50~60%）                 │
│    长期持有，减少交易频率                   │
├──────────────────────────────────────────┤
│  AI算力龙头    20~25%                      │
│  新能源龙头    15~20%                      │
│  高股息红利    15%                         │
├──────────────────────────────────────────┤
│          卫星仓位（20~30%）                 │
│     灵活轮动，捕捉阶段性机会              │
├──────────────────────────────────────────┤
│  AI应用/软件   10%   (科技补涨轮动)        │
│  周期资源品    10%   (涨价趋势跟踪)         │
│  消费/医药     10%   (政策催化博弈)        │
├──────────────────────────────────────────┤
│          现金仓位（10~20%）                 │
│    应对系统性风险和黑天鹅                  │
└──────────────────────────────────────────┘
```

### 关键操作纪律
1. **不追高**：单日涨幅 >7% 时不加仓，等回调
2. **分批建仓**：任何标的至少分 3 次买入
3. **季度再平衡**：每季度末检视一次仓位比例
4. **止损纪律**：个股跌破成本线 15% 重新评估逻辑
