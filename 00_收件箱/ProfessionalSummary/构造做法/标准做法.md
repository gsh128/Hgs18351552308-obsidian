# 地产标准做法2026

## 地上部分
### 屋面构造做法
- **住宅，配套屋面（上人/不上人、有保温）平屋面1**  
	1. 钢筋混凝土屋面板，表面清理干净（结构自防水渗漏检查处理）（抗渗等级P6级）
	2. 1.5厚聚合物水泥防水涂料，遇墙处上翻高出完成面300，管根及阴角处加设加强层
	3. 5厚M15聚合物水泥砂浆找平层
	4. 1.5厚非固化橡胶沥青防水涂料，遇墙处上翻高出完成面300，管根及阴角处加设加强层
	5. 1.5厚自粘聚合物改性沥青防水卷材（高分子膜基），遇墙处上翻高出完成面300，端头采用30宽2厚铝型材压条通长设置，螺丝@200固定
	6. 最薄处30厚泡沫混凝土（ρ=600kg/m³）2%找坡，随捣随抹平压光
	7. 105厚聚苯乙烯挤塑聚苯板（XPS）保温层（燃烧性能等级为B1级，X350）
	8. 50厚C30细石混凝土，内配Φ4@200双向钢筋网，随捣随抹光，设6m×6m分格缝，缝宽20mm，缝内嵌沥青油膏密封材料
- **楼梯间、电梯间屋面（不上人，无保温）平屋面2**  
	1. 钢筋混凝土屋面板，表面清理干净
	2. 1.5厚聚合物水泥防水涂料，遇墙处上翻高出完成面300，管根及阴角处加设加强层
	3. 5厚M15聚合物水泥砂浆找平层
	4. 1.5厚非固化橡胶沥青防水涂料，遇墙处上翻高出完成面300，管根及阴角处加设加强层
	5. 1.5厚自粘聚合物改性沥青防水卷材（高分子膜基），遇墙处上翻高出完成面300，端头采用30宽2厚铝型材压条通长设置，螺丝@200固定
	6. 最薄处30厚泡沫混凝土（ρ=600kg/m³）2%找坡，随捣随抹平压光
	7. 50厚C30细石混凝土，内配Φ4@200双向钢筋网，随捣随抹光，设6m×6m分格缝，缝宽20mm，缝内嵌沥青油膏密封材料
- **出地面风井顶板 / 出屋面设备井顶板平屋面3（不上人，无保温）**
	1. 钢筋混凝土屋面板，表面清理干净
	2. 1.5厚聚合物水泥防水涂料，墙角及有垂直管道处翻高300
	3. 最薄20厚DS M20砂浆（1:2.5水泥砂浆）抹平压光，2%找坡
	4. 普通外墙涂料，与外墙材料同色
- **混凝土雨棚平屋面4（不上人，无保温）**
	1. 钢筋混凝土屋面板，表面清理干净
	2. 1.5厚聚合物水泥防水涂料，墙角及有垂直管道处翻高300
	3. 10厚水泥基无机保温砂浆
	4. 最薄处20厚C20细石混凝土，找坡1—2%压光，向地漏或者最低点找坡
	5. 普通外墙涂料，与外墙材料同色
- **屋面构架**
	- 外墙涂料，与外墙材料同色（屋面构架外侧）
	- 普通外墙涂料，与外墙材料同色（屋面构架内侧、底面、顶面）

### 外墙构造做法

- **公建外墙 / 住宅砂胶砖外墙**
	- 有保温要求做法（住宅外墙常规做法）**
		1. 基层墙体清理干净，界面剂一道
		2. 砖墙与混凝土墙交界处挂300mm宽钢丝网，每边搭接150mm，首层、顶层满铺耐碱玻璃纤维网格布
		3. 15厚DP M15砂浆（1:3水泥砂浆）（掺避水胶）找平层
		4. 1.5厚聚合物水泥防水涂料
		5. 3厚专用粘接剂
		6. 50厚复合发泡水泥板（A级），锚栓固定，热镀锌角钢托架每层楼层位置及门窗洞口上方设置（厚度详节能专篇）
		7. 3厚聚合物抗裂砂浆（压入第一层耐碱玻璃纤维网格布增强网，专用锚固件锚固）
		8. 2厚聚合物抗裂砂浆（压入第二层耐碱玻璃纤维网格布增强网，专用锚固件锚固）
		9. 两遍外墙腻子，打磨平整
		10. 外墙涂料，颜色详立面图
	- **无保温要求做法**
		- *取消做法5、6、8*（即无保温层及部分抗裂层）
- **干挂面层做法**
	1. 基层墙体清理干净，界面剂一道
	2. 砖墙与混凝土墙交界处挂300mm宽钢丝网，每边搭接150mm
	3. 10厚DP M15砂浆（1:3水泥砂浆）（掺避水胶）找平层
	4. 1.5厚聚合物水泥防水涂料
	5. 3厚专用粘接剂
	6. 干挂铝板/石材（岩棉条保温），厚度见节能专篇，专用锚固件锚固，燃烧性能为A级
- **一体板面层做法**
	1. 基层墙体清理干净，界面剂一道
	2. 砖墙与混凝土墙交界处挂300mm宽钢丝网，每边搭接150mm
	3. 10厚DP M15砂浆（1:3水泥砂浆）（掺避水胶）找平层
	4. 1.5厚聚合物水泥防水涂料
	5. 3厚专用粘接剂
	6. 石材/铝板保温一体板（岩棉条保温），厚度见节能专篇，专用锚固件锚固，燃烧性能为A级

### 楼面构造做法

一、起居室、餐厅、户内公共走道、卧室、书房、储藏间、衣帽间楼面 (H-0.100)

1. 现浇钢筋混凝土楼板，随捣随抹平，表面清理干净
2. 隔声减振垫层（根据施工招标选用材料）：
   - 13厚K/隔声保温减振垫层 (L1a)
   - 8厚FC保温隔声减振垫层 (L1b)
   - 8厚DZ保温隔声减振垫层 (L1c)
3. 保温隔声砂浆层：
   - 40厚K/隔声保温砂浆 (L1a)
   - 50厚FC保温隔声砂浆 (L1b)
   - 45厚DZ保温隔声砂浆 (L1c)
4. 面层（装修时施工）

---

二、住宅卫生间楼面 (2F及以上 H-0.170 / 首层 H-0.120)

*注：3-6层做法结合个性化装修实施*

1. 现浇钢筋混凝土楼板，随捣随抹平，表面清理干净
2. 1.5厚聚合物水泥防水涂料，墙角及有垂直管道处翻高300
3. 80厚LC7.5轻骨料混凝土（ρ<800kg/m³）保温填充层（首层无此做法）（个性化装修时施工）
4. 最薄处20厚C20细石混凝土找坡层，向地漏找1%坡，随打随抹平。地漏四周及管根部用DS M15砂浆（1:3水泥砂浆）抹小八字角（个性化装修时施工）
5. 1.5厚聚合物水泥防水涂料，墙角及有垂直管道处翻高300（个性化装修时施工）
6. 面层（个性化装修时施工）

---

2. 1.5厚聚合物水泥防水涂料，墙角及有垂直管道处翻高300
3. 隔声减振垫层（根据施工招标选用材料）：
   - 13厚K/隔声保温减振垫层 (L1a)
   - 8厚FC保温隔声减振垫层 (L1b)
   - 8厚DZ保温隔声减振垫层 (L1c)
4. 保温隔声砂浆层：
   - 40厚K/隔声保温砂浆 (L1a)
   - 50厚FC保温隔声砂浆 (L1b)
   - 45厚DZ保温隔声砂浆 (L1c)
5. 1.5厚聚合物水泥防水涂料，墙角及有垂直管道处翻高300
6. 面层（个性化装修时施工）
