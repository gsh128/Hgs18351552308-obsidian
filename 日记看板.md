# 🗂️ 日记看板

> 📥 自动从 **[[DailyNote]]** 日记中抓取四大板块内容，按日期倒序排列

```dataviewjs
// ============================================================
// 日记看板 - 通过解析文件内容按标题提取板块
// ============================================================

const sectionConfig = [
  { name: "今日学习", emoji: "📚" },
  { name: "今日想法", emoji: "💭" },
  { name: "今日反思", emoji: "🔄" },
  { name: "重要提醒", emoji: "⚠️" },
];

const pages = dv.pages('"10_日记"').sort(p => p.file.name, 'desc');
dv.paragraph("📊 找到 `" + pages.length + "` 篇日记");

for (const config of sectionConfig) {
  const items = [];

  for (const page of pages) {
    const path = page.file.path;
    const content = await dv.io.load(path);
    if (!content) continue;

    // 按二级标题分割，找到对应板块
    const sections = content.split(/^## /m);
    for (const section of sections) {
      const lines = section.split('\n');
      const heading = lines[0].trim();
      if (heading !== config.name) continue;

      // 提取该板块下的列表项（跳过空行和标题行）
      for (let i = 1; i < lines.length; i++) {
        const line = lines[i].trim();
        if (line.startsWith('- ') && line.length > 2) {
          // 去掉 emoji 前缀（如 📚、💭）让输出更干净
          const cleanText = line.slice(2).replace(/^[📚💭🔄⚠️]\s*/, '');
          items.push(`[[${page.file.name}]] · ${cleanText}`);
        }
      }
    }
  }

  if (items.length > 0) {
    dv.header(3, config.emoji + " " + config.name + "（共 " + items.length + " 条）");
    dv.list(items);
    dv.paragraph("---");
  }
}
```


## 📌 使用说明

- ✅ **自动更新** — 每次打开/刷新此笔记，Dataview 会自动从 `10_日记/` 文件夹读取所有日记
- ✅ **点击跳转** — 每条记录前的日期链接可直接点击跳转到原始日记
- ✅ **新增日记** — 只需在日记模板的对应板块下写列表项，内容会自动出现在这里
- 🔧 **修改模板** — 详见 [[Template/DailyNoteTemplate.md]]

## 🔗 快速跳转

```dataviewjs
const latest = dv.pages('"10_日记"').sort(p => p.file.name, 'desc')[0];
if (latest) {
  dv.paragraph("- [[Template/DailyNoteTemplate.md|📋 日记模板]]");
  dv.paragraph("- [[" + latest.file.name + "|📝 最新日记：" + latest.file.name + "]]");
} else {
  dv.paragraph("- [[Template/DailyNoteTemplate.md|📋 日记模板]]");
}
```
