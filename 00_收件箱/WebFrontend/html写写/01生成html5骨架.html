生成 HTML 骨架

📝

生成 HTML5 骨架

1. 英文模式下输入 ! 后，按 Tab 键自动生成

2. 检查文件格式是否正确或已保存

3. 在 VSCode 中使用快捷生成

🌐

预览方式 - 直接打开

1. 双击文件夹中的网页图标查看

2. 推荐 Chrome 浏览器，设为默认

3. 修改后手动刷新浏览器更新

⚡

预览方式 - Live Server

1. 安装 Live Server 插件实现热更新

2. Ctrl+Shift+P 选择 Open With Live Server

3. 需将网页存放在已打开的文件夹中

4. 分屏模式开发，所见即所得

🏗️

HTML5 骨架结构说明

DOCTYPE：声明文档类型，告诉浏览器用 HTML5 标准解析

html：根元素，lang 属性设置页面语言（zh-CN）

head：元数据区，包含字符编码、视口、标题等

body：页面内容区，用户可见的内容都写在这里

meta charset：设置字符编码为 UTF-8，防止乱码

meta viewport：适配移动端显示

title：网页标题，显示在浏览器标签页
