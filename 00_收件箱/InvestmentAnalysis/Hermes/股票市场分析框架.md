## 一、市场研判五维框架

### 1. 政策面 🏛️
- **宏观政策**：货币政策（降准降息/LPR）、财政政策（专项债/特别国债）、两会/政治局定调
- **产业政策**：AI、新能源、半导体、生物医药、低空经济等行业扶持文件
- **监管动向**：证监会/交易所新规、减持限制、IPO节奏
- **关键信息来源**：新华社、央行网站、各部委官网

| 日期 | 政策事件 | 影响板块 | 力度评级 | 备注 |
|------|----------|----------|----------|------|
| | | | ⭐⭐⭐⭐⭐ | |

### 2. 资金面 💰
- **北向资金**：日度/周度净流入流出，重点买入/卖出行业
- **两融余额**：融资余额趋势、融券余额变化
- **公募/私募仓位**：股票型基金仓位、新发基金规模
- **成交量/成交额**：两市成交额、量比
- **关键信息来源**：东方财富、Wind、同花顺、天天基金

| 日期 | 两市成交额 | 北向净流入 | 融资余额 | 情绪判断 |
|------|-----------|-----------|----------|----------|
| | 亿 | 亿 | 亿 | 活跃/正常/低迷 |

### 3. 估值面 📊
- **市盈率分位数**：各指数 PE-TTM 历史分位（沪深300/中证500/创业板）
- **破净率**：破净股票占比
- **股债性价比（ERP）**：股权风险溢价 = 1/PE - 国债收益率
- **行业估值对比**：高估/低估行业梳理
- **关键信息来源**：理杏仁、芝士财富、Wind

| 日期 | 沪深300 PE分位 | 中证500 PE分位 | ERP | 估值判断 |
|------|---------------|---------------|-----|----------|
| | % | % | % | 偏贵/合理/低估 |

### 4. 情绪面 😱😎
- **涨停/跌停家数**：涨停>100/跌停>10 等关键阈值
- **连板高度**：最高连板数，反映短线情绪
- **恐慌指数**：中国波指（iVIX）、期权PCR
- **社交媒体热度**：雪球/淘股吧热搜、散户情绪调查
- **关键指标**：恐贪指数（韭圈儿）

| 日期 | 涨停家数 | 跌停家数 | 连板高度 | 恐贪指数 | 情绪判断 |
|------|----------|----------|----------|----------|----------|
| | | | | | 恐惧/中性/贪婪 |

### 5. 基本面 📈
- **宏观经济**：GDP增速、PMI、CPI/PPI、社融/M2
- **企业盈利**：A股整体盈利增速、ROE趋势
- **行业景气度**：新能源车销量、半导体出货量、光伏装机等
- **关键信息来源**：国家统计局、Wind、券商研报

| 日期 | PMI | CPI | 社融 | GDP预期 | 基本面判断 |
|------|-----|-----|------|---------|-----------|
| | | % | 万亿 | % | 扩张/收缩/平稳 |

## 二、核心主线追踪

### AI 产业链 🤖
- **算力层**：GPU/服务器 → 光模块 → 数据中心/液冷
- **模型层**：大模型能力迭代、国产替代（华为昇思等）
- **应用层**：AI Agent、AI+医疗、AI+教育、AI+制造
- **关注标的**（自行补充）：[[AI概念股跟踪]]

### 新能源 ⚡
- **固态电池**：量产进度、产业链配套
- **智能驾驶**：L3政策放开、城市NOA渗透率
- **光伏/储能**：产能出清节奏、海外需求
- **关注标的**（自行补充）：[[新能源概念股跟踪]]

### 高股息红利 💎
- **银行**：国有大行、股份行
- **煤炭/石油**：高分红央企
- **公用事业**：电力、高速、水务
- **选股标准**：股息率>4%、经营现金流稳定、央企/国企
- **关注标的**（自行补充）：[[红利股跟踪]]

### 国产替代/自主可控 🛡️
- **半导体**：设备、材料、EDA、先进封装
- **信创**：操作系统、数据库、办公软件
- **军工**：航空航天、信息化
- **关注标的**（自行补充）：[[国产替代概念股跟踪]]

### 消费复苏 🛍️
- **必选消费**：食品饮料、调味品
- **可选消费**：旅游、免税、家电（以旧换新政策）
- **新消费**：谷子经济、潮玩、宠物
- **关注标的**（自行补充）：[[消费概念股跟踪]]
