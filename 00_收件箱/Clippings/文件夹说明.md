# Clippings - 收藏夹说明

## 文件夹用途

此文件夹用于收藏和保存以下类型的网络资源：

- 📄 **网页文章** - 值得阅读和收藏的在线文章、博客文章、教程等
- 📹 **视频内容** - 教程视频、纪录片、课程录像等
- 🔗 **网页书签** - 需要后续参考或访问的重要网页链接

## 文件命名建议

建议采用清晰的命名格式，便于日后检索：

- 使用描述性标题
- 包含来源或作者信息（如需要）
- 添加日期标记（可选）

示例：
- `Claude Code 使用指南.md`
- `Python 高级编程教程.md`
- `2024-04-20 - 网络安全入门.md`

## 内容组织

随着收藏内容增多，可以考虑建立子文件夹进行分类：

```
Clippings/
├── 技术文章/
├── 视频教程/
├── 行业资讯/
└── 趣味内容/
```

## 使用方式

1. 将网页内容复制或导入到此文件夹
2. 使用 Markdown 格式保存，便于在 Obsidian 中查看和编辑
3. 利用 Obsidian 的标签系统进行分类（如 `#技术`、`#教程`）
4. 使用双向链接关联相关的收藏内容
5. 定期整理和归档，删除不再需要的内容

## 相关功能

- 配合 Obsidian 的 **Dataview** 插件，可以快速按标签或日期筛选收藏
- 使用 **Graph View** 可以查看收藏内容之间的关联关系
- 通过 **Search** 功能快速定位需要的文章或视频
