认识 HTML

了解网页超文本标记语言的基本概念、结构组成及其在 Web 开发中的重要地位。

一、什么是 HTML

HTML（HyperText Markup Language）即超文本标记语言，是用于描述网页结构的标准标记语言。它不是一种编程语言，而是一种标记语言，用于告诉浏览器如何结构化页面内容。

核心特点

  • 超文本：不仅能展示文本，还能包含链接、图片、视频、音频等多种媒体内容
  • 标记语言：使用标签（Tag）来标识和描述文档的不同部分
  • 跨平台：任何支持浏览器的设备都可以解析和显示 HTML 页面
  • 易于学习：语法简单直观，是 Web 开发的入门基础

二、HTML 的基本结构

一个标准的 HTML5 文档由声明、根元素、元数据区和内容区四部分组成。这些部分共同构成了网页的完整骨架。

结构组成

<!DOCTYPE html> <html> <head> <body>
  • <!DOCTYPE html>：文档类型声明，告诉浏览器使用 HTML5 标准解析
  • <html>：根元素，包含 lang 属性设置页面语言（zh-CN）
  • <head>：元数据区，包含字符编码、视口配置、页面标题等
  • <body>：内容区域，用户可见的所有页面内容都写在这里

三、常见 HTML 标签

文本标签

  • <h1> ~ <h6>：标题标签，h1 最大，h6 最小
  • <p>：段落标签，用于包裹文本内容
  • <span>：行内容器，用于文本样式的局部应用
  • <a>：链接标签，创建超链接

容器标签

  • <div>：块级容器，用于布局分区
  • <header>：页头或区块头部
  • <footer>：页脚或区块底部
  • <section>：文档章节

列表标签

  • <ul>：无序列表
  • <ol>：有序列表
  • <li>：列表项

多媒体标签

  • <img>：插入图片
  • <video>：插入视频
  • <audio>：插入音频
